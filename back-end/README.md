## Quick Setup

From the `backend` directory follow these steps:

### Step 1: Install Dependencies

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Step 2: Set Up Environment Variables

Create a `.env` file:

```
# Required API Keys
GOOGLE_API_KEY=
# Server Configuration
MONGODB_URL=
DATABASE_NAME=
```

### Step 3: Start Server

```bash
python main.py
```
